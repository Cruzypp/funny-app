# Vela App (iOS)

App iOS creada durante un hackathon con enfoque en **Human-Centered AI (IA centrada en las personas)**.  
El objetivo principal es **mejorar la seguridad de los transeúntes** mediante asistencia inteligente y recomendaciones contextuales, priorizando una experiencia clara, útil y respetuosa con la privacidad.

## Enfoque: Human-Centered AI
Este proyecto se diseñó bajo principios de IA centrada en humanos:

- **Seguridad y bienestar primero**: la app busca ayudar a reducir riesgos para peatones/transeúntes.
- **Transparencia**: el usuario debe entender qué hace la app y por qué.
- **Control del usuario**: el usuario decide cuándo activar funciones y qué permisos otorgar.
- **Privacidad por diseño**: minimización de datos y uso responsable de sensores/información.

## Problema que busca resolver
En entornos urbanos, los transeúntes se enfrentan a riesgos como distracciones, zonas peligrosas, cruces complejos o falta de visibilidad.  
La app busca **apoyar la toma de decisiones** del usuario en situaciones cotidianas, reduciendo la probabilidad de incidentes.
